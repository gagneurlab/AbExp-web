This website provides an interface for running AbExp on arbitrary variants. AbExp is a method that predicts aberrant gene expression across human tissues (HÃ¶lzlwimmer et al., bioRxiv) by integrating various variant annotations with tissue-specific isoform proportions and expression variability. The predicted AbExp score shows how much the gene expression is above or below the average population, measured in standard deviations.
